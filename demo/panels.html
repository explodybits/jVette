jVette Panel(s)

These pages are live examples of jVette; click the header icons (above) to see it in action! View the source code to see actual markup/attribute implementation details.

Add the data attribute data-jv-panel with a value of left | right to define entire panel(s), rather than identifying individual content blocks.

<div data-jv-panel="left">Left</div>
<div data-jv-panel="right">Right</div>
